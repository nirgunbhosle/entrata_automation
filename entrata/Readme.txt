Project Structure
src/main/java: Contains the source code files and pages of page object for the automation tests.
src/test/java: Includes the test scripts and scenarios.
stc/test/resources: Includes all the feaure files, config and drivers. 
Target: Contains generated test reports.


Setup Instructions
Prerequisites: Ensure that you have the following installed:

Java Development Kit (JDK)
Eclipse IDE
Required dependencies and libraries
Cloning the Repository: Clone this repository to your local machine using Git:

Importing Project into Eclipse:

Open Eclipse IDE.
Navigate to File > Import.
Select Existing Projects into Workspace and click Next.
Choose the root directory of the cloned repository and click Finish.
Configuring Dependencies: Make sure all required dependencies and libraries are properly configured in the project build path.

Usage
Running Tests:

Right-click on the test suite or individual test cases.
Select Run As > Junit Test.
Generating Reports:

Test reports are generated automatically after test execution.
Reports can be found in the Cucumber Reports under target directory.
